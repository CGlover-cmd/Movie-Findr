Movie Findr is a free, open-source, non-commercial web browser game built as a hobby project and hosted on GitHub Pages. It functions similarly to the “Wikipedia Game” but utilizes movie casts. The game challenges players to find a connecting path between a randomly selected starting movie and a target movie.
To achieve this, the application relies on the TMDb API to fetch data and images. Specifically, it uses the /movie/popular endpoint to generate the starting and target movies, the /movie/{movie_id}/credits endpoint to display the cast of a selected film, and the /person/{person_id}/movie_credits endpoint to display the filmography of a selected actor. The application is completely free to play, contains zero advertisements, and does not generate any revenue, strictly adhering to TMDb’s non-commercial usage terms.
